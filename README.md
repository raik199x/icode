— Хранить данные можно в любой БД.

— Использовать ООП подход для сущностей Студент, Преподаватель, Предмет и Группа.

— Для сущностей Студент, Преподаватель, Предмет и Группа реализовать пункты:

    просмотр записей

    удаление записей
